# Student Grade Calculator

## Project Overview
This Python program calculates a student's grade based on marks
and displays encouraging messages.

## Features
- Uses if-elif-else for grading
- Input validation (0–100 only)
- Uses while loop for error handling
- Uses function for grade calculation

## Grading Logic
A: 90–100  
B: 80–89  
C: 70–79  
D: 60–69  
F: 0–59  

## How to Run
1. Install Python 3
2. Open terminal in project folder
3. Run:
   python grade_calculator.py
